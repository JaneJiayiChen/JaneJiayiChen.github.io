## 概述
MRONJ 的手术预后差异明显，临床上常需要借助影像线索判断手术时机与方案。本项目聚焦一个关键问题：**哪些影像学特征与术后预后真正相关**，这些关联能否为临床筛查和决策提供支持？我们选取三项常见影像特征，在单变量与多变量框架下进行系统检验。

为保证解释的严谨性，本项目采用**假设检验**的统计框架，通过 Logistic 回归分别评估“单变量信号”与“联合检验后的信号”。结果均为聚合输出，用于支持思考与讨论，而非直接给出确定性预测。

## 研究问题与假设
我们通过一系列假设检验来回答“影像学特征是否与预后相关”。

- 对每个变量 \\(X_j\\)，检验 \\(H_0: \\beta_j = 0\\)（Wald 检验）。
- 对多分类变量，使用似然比检验（LLR）判断整体显著性。

## 结局与自变量
- 结局：\\(Y=1\\) 为 **1A有效**，\\(Y=0\\) 为 **2B/C/D/E**。
- 影像学自变量：
  - 骨膜反应（1是2否）
  - 死骨分离（1是2否）
  - 病变特点（1破坏2硬化3混合）

## 统计模型
二元 Logistic 回归：

\\[
\\log\\frac{P(Y=1\\mid X)}{1-P(Y=1\\mid X)} = \\beta_0 + \\sum_j \\beta_j X_j
\\]

对每个 \\(X_j\\) 进行 Wald 检验，多变量影像模型用 LLR 评估整体效应。

## 结果快照（聚合）
下表给出模型整体表现与各影像特征的显著性证据。

**模型指标**

| 指标 | 数值 |
| --- | --- |
| AUC | 0.730 |
| Accuracy | 0.731 |
| Sensitivity | 0.928 |
| Specificity | 0.328 |
| Precision | 0.739 |
| F1 | 0.823 |
| 阈值（准确率优化） | 0.344 |
| 影像多变量 LLR p 值 | 0.0678 |
| Pseudo R\\(^2\\) | 0.037 |

**影像学单因素检验**

| 特征 | 检验 | P 值 | 方向 |
| --- | --- | --- | --- |
| 骨膜反应（1是2否） | Wald | 0.9730 | 负向 |
| 死骨分离（1是2否） | Wald | 0.0479 | 正向 |
| 病变特点（1破坏2硬化3混合） | LLR | 0.0260 | Overall |

## 关键结论
**死骨分离** 与 **病变特点** 在单因素层面具有显著信号（P < 0.05），但在三项特征联合检验后整体信号趋于边缘（LLR p=0.0678），提示其“独立效应”证据有限。预测模型采用类权重与正则化后，准确率提升至 0.731，但在当前阈值下更偏向高敏感度。

## 讨论（假设检验视角）
本项目的结果更像是“有信号但不够稳健”的模式。影像学特征彼此相关时，单因素显著往往会在联合检验中被稀释。此外，稀疏类别与完全病例分析会减少有效样本量，使统计功效下降、区间变宽。预测模型通过类权重与正则化缓解了不平衡问题，但阈值选择仍决定了敏感度与特异度的取舍。

## 医学解释与启示
从医学实践看，**死骨分离** 与 **病变特点** 的显著性符合临床直觉：清晰的死骨分离与病变形态通常反映病变阶段与生物学行为。**骨膜反应** 未见显著信号，可能意味着其增量信息有限或存在测量波动。总体而言，影像学特征更适合作为**筛查或风险分层**工具，而不是单独用于确定性预后判断。

## 实践启示
短期内，影像特征可用于**优先级排序**，帮助快速识别高风险患者；方法学上需要提升数据完整性、合并稀疏类别、优化阈值；临床上建议标准化影像标注以降低噪声、提升可重复性。

## 下一步
- 提升数据完整性以纳入混杂因素。
- 合并稀疏类别以减少分离问题。
- 优化阈值并在新数据上验证。
